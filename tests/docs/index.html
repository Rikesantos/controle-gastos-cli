Controle de Gastos CLI

Aplicação em Python para controle de gastos pessoais via linha de comando.

Entrega Intermediária

Nesta etapa foi integrada a API pública AwesomeAPI para consultar a cotação do dólar em tempo real.

Como executar

pip install -r requirements.txt
python src/app.py
